# -INTERACTIVE-QUIZ-APPLICATION

An interactive and responsive quiz application that allows users to test their knowledge through multiple-choice questions.
It features real-time score tracking, question navigation, instant feedback, a results summary, and a light/dark theme for an engaging user experience.

Technologies Used
HTML5
CSS3
JavaScript (ES6)
Font Awesome
Responsive Web Design
